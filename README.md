### [并查集 Disjoint Set Union](./data_structure/DisjointSetUnion.cpp)
并查集是一种树形的数据结构支持 **查找(Find)** 和 **合并(Union)** 操作。 

**查找(Find)**: 判断两个元素是否在同一个集合内/是否有同一个祖先。

**合并(Union)**：合并两个集合/类别。
